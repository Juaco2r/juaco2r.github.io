Cell Well Segmentation & Feature Extraction

Classical computer vision pipeline for instance segmentation and quantitative feature extraction of low-density cell wells from multi-channel fluorescence microscopy TIFF images (no ML/DL dependencies).

Python ≥ 3.9 Watershed Multi-channel TIFF QuPath GeoJSON
Repository Docs

Key features

  • Multi-channel TIFF handling
  • Nuclei-based seed detection
  • Marker-controlled watershed instance segmentation
  • Morphological + intensity feature extraction
  • CSV per-cell export
  • QuPath-compatible GeoJSON annotations
  • QC visualization preview
:contentReference[oaicite:18]{index=18}

Input format

Input: multi-channel fluorescence microscopy images stored as TIFF (.tif/.tiff).

Channel convention

  • 0: Nuclear stain (e.g., DAPI)
  • 1: Marker channel (red)
  • 2: Marker channel (green)
  • 3: Cytoplasmic stain (CellCyto)

If your channel order differs, update the mapping in create_channel_images() within pipeline.py. :contentReference[oaicite:19]{index=19}

Run (CLI)

Recommended usage is through the command-line interface:

python -m cellwell.cli --input "path/to/your/data/*.tif"

For each input file, an output folder is created automatically. :contentReference[oaicite:20]{index=20}

Outputs

For each processed image, typical outputs include:

  • RGB.tif (RGB composite)
  • CellCyto.tif (composite used for segmentation)
  • instances.tif (uint16 instance labels)
  • cell_features.csv (per-cell measurements)
  • qupath_final.geojson (QuPath polygons)
  • preview.png (QC summary)

Method summary

  1. Gaussian smoothing of nuclear channel
  2. Local maxima detection for seeds
  3. Otsu thresholding for foreground detection
  4. Marker-controlled watershed
  5. Post-filtering by object area

Feature extraction is performed on original-resolution RGB data to preserve intensity fidelity. :contentReference[oaicite:22]{index=22}

Notes & limitations

  • Optimized for low-density cell wells.
  • Parameters may need tuning for different magnifications/stains.
  • Not intended for highly confluent/overlapping cells.
  • No ML/DL models are used.
